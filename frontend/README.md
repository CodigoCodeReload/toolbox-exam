# Files Processor Frontend

Este es un frontend desarrollado en React para consumir la API de Files Processor.

## Requisitos

- Node.js 16+

## Instalación

```bash
npm install
```

## Ejecución en desarrollo

```bash
npm start
```

La aplicación se abrirá automáticamente en tu navegador en http://localhost:3002

## Compilación para producción

```bash
npm run build
```

Esto generará una carpeta `dist` con los archivos optimizados para producción.

## Tecnologías utilizadas

- React 18
- React Bootstrap
- Axios para peticiones HTTP
- Webpack para la compilación
